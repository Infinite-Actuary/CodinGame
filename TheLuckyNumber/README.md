How many lucky numbers with n or less digits? i.e. number of lucky numbers between 0 - 10^n inclusive?

`L_n = 2 * (9^n - 8^n)` = {0, 2, 34, 434, 4930, ...}

e.g. There are 434 lucky three digit numbers _ _ _.
